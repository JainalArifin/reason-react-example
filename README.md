# ReasonReact

ReasonReact documentation [here](https://reasonml.github.io/reason-react/).

## Run Project

```sh
npm install
npm run server
# in a new tab
npm start
```

Open a new web page to `http://localhost:8000/`. Change any `.re` file in `src` to see the page auto-reload.
